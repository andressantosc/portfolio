Front-End Web Developer Nanodegree

In this Nanodegree you learn the fundamentals of how the web works and gain a working knowledge of the three foundational languages that power each and every website: HTML, CSS and JavaScript. You learn to build beautiful, responsive websites optimized for security and performance. You build portfolio-worthy projects for which you receive expert feedback on your code.

You are provided with visual assets and a game loop engine; using these tools I created a number of entities to the game including the player characters and enemies to recreate the classic arcade game Frogger. I used object-oriented javascript.

Interactive resume application that reads the resume content from a JSON file and dynamically displays that content within a provided template. I used objects, functions, conditionals, and control structures to compose the content that will display on the resume.
